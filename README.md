

# Miitrix-NX

Miitrix-NX is a fork of [miitrix](https://github.com/Sorunome/miitrix/) for the Nintendo Switch. It requires [matrix-nx-sdk](https://github.com/RoblKyogre/matrix-nx-sdk/) to be compiled.
